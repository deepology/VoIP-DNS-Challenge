# VoIP-DNS-Challenge

## Test Partitions

- `src` = Source Audio
- `nrm` = Normalized Audio
- `zm`  = Zoom
- `gm`  = Google Meets
- `cloud` = Audio Relayed To Cloud.
- `phone` = Audio Relayed To Phone.
- `low` = Low Denoising Setting
- `auto` = Auto Denoising Settting
